### Minesweeper Game

#### Instructions

- Make sure docker is up and running.
- Run `./cmd.sh composer install` to install all dependencies 
- Run `./cmd.sh php play.php` to start playing the game.
- Run `./cmd.sh vendor/bin/phpunit test.php` to run the test cases
